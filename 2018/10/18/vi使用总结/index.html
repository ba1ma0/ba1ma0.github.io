vi使用总结

发表于 2018-10-18 | 分类于 技巧总结 | | 本文总阅读量 人次
字数统计: 397 | 阅读时长 ≈ 1

前言：

之前总结的vi使用技巧现在拿出来分享一下

vi是linux上非常强大的编辑器，之前专门花时间学习了一下并且总结了一下

  1. 如何进入编辑模式
    Insert 按键即可进入编辑模式

  2. 如何进入命令行模式
    Esc 键即可进入命令行模式

  3. 如何设置每次打开都能显示行号
    1.进入配置文件etc/vim //这里要设置一个配置文件,每次打开vi都要加载
    2.vim ~/.vimrc
    3.set number //也就是加入一些常用的设置

  4. 一些常用的命令
    :w //写
    :wq //保存退出
    :set number //显示行号
    :set nonu //隐藏行号
    :u //撤销上一步操作
    删除
    :.,$d //从最后一行到光标所在的一行全部删除
    :11,13d //从11行到13行全部删除
    复制
    yy //复制光标在的那行
    3yy //复制三行光标所在行+下面的两行
    y$ //复制光标所在行到行尾
    y^ //复制光标所在行到行头
    黏贴
    p //将所复制的内容黏贴到光标的下面
    剪切
    dd //剪切一行,切记这里没有冒号
    3dd //剪切三行光标所在行+下面的两行
    d$ //剪切光标所在行到行尾
    d^ //剪切光标所在行到行头

(5)对鼠标的一些操作
Ctrl+u //向上翻半屏
Ctrl+f //向上翻一屏
Ctrl+d //向下翻半屏
Ctrl＋b //向下翻一屏

上面总结的基本上已经可以满足我的绝大多数需求
